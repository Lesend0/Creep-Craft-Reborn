# 🟩 Creep Craft: Reborn

**Creep Craft: Reborn** — это масштабная 2D-песочница с элементами выживания, вдохновленная Minecraft и является прямым ремейком старой флеш игры Creep Craft. Игра работает прямо в браузере и написана полностью на Vanilla JavaScript, HTML5 Canvas и CSS3 без использования сторонних игровых движков или библиотек.

## 📖 Сюжет (Story Mode)
Вы играете за крипера, который оказался в мире Стива. Сам Стив погиб в хардкорном режиме и больше не возродится. Ваша задача — завершить игру за него, выжить его методами и в конечном итоге уничтожить дракона.

## 🌟 Ключевые особенности

* **Процедурная генерация мира:** Мир размером 20 000 на 256 блоков с биомами, пещерами, озерами воды и лавы, а также генерацией различных руд (уголь, железо, золото, алмазы).
* **Сложные игровые механики:** 
  * Полноценная система крафта (Crafting Table) и переплавки руд (Furnace).
  * Физика текучих жидкостей (вода и лава) с 8 уровнями растекания.
  * Механики распространения огня и взрывы TNT.
  * Земледелие (вспашка земли мотыгой, посадка семян, рост пшеницы).
* **Динамическое окружение:** 
  * Плавная смена дня и ночи (Day/Night cycle) с изменением палитры неба и появлением звезд.
  * Продвинутая система освещения (Lightmap) — свет от факелов, лавы и активных печей реалистично рассеивается в темноте.
  * Система частиц (Particle System v2) для визуальных эффектов (осколки блоков, кровь, искры, листопад, дым).
* **Сущности и AI:**
  * Враждебные мобы (Зомби, Пауки) с продвинутым поиском пути.
  * Мирные мобы (Коровы, Свиньи, Овцы) с различными состояниями (любопытство, поедание травы, паника).
* **Кастомный Аудио-движок:** Звуки и музыка генерируются процедурно с помощью Web Audio API (синусоиды, шум, фильтры). Реализовано 3D-позиционирование звука для шагов мобов и эмбиента пещер.
* **Богатый интерфейс:**
  * Инвентарь с поддержкой Drag & Drop, сундуки (включая двойные), интерфейс печи.
  * Внутриигровая Энциклопедия команд (Commands Encyclopedia), система достижений (Achievements) и Чат.

## 🛠 Технологии

* **JavaScript (ES6+)** — вся игровая логика, физика, ИИ и обработка коллизий.
* **HTML5 Canvas** — рендеринг графики (включая кэширование тайлов для оптимизации).
* **CSS3** — стилизация UI-элементов (меню, HUD, инвентарь) в пиксельном стиле.

## 🚀 Как играть

Игра не требует установки, серверов или баз данных.

1. Склонируйте репозиторий:
   \`\`\`bash
   git clone https://github.com/ВАШ_НИК/creepcraft-reborn.git
   \`\`\`
2. Перейдите в папку проекта и просто откройте файл `index.html` в любом современном веб-браузере.
   *Примечание: для лучшей производительности рекомендуется использовать Google Chrome или браузеры на базе Chromium.*

## 🎮 Управление

* **W, A, S, D / Стрелки** — Перемещение
* **Space / W / Вверх** — Прыжок
* **ЛКМ (Левая кнопка мыши)** — Атака мобов / Добыча блоков
* **ПКМ (Правая кнопка мыши)** — Размещение блоков / Взаимодействие (съесть еду, разлить ведро)
* **E** — Открыть инвентарь
* **Q** — Выбросить предмет (когда открыт инвентарь)
* **1-9 / Колесико мыши** — Выбор предмета в хотбаре
* **T / \` / /** — Открыть внутриигровой чат / ввод команд
* **F3** — Экран отладки (FPS, координаты)
* **Esc** — Пауза / Меню настроек

## ⌨️ Команды (Чат)
В игре поддерживается множество чит-команд. Вот некоторые из них:
* `/time day` или `/time night` — Сменить время суток.
* `/give [item_name] [count]` — Выдать предмет.
* `/summon [zombie/spider/pig/cow/sheep]` — Заспавнить моба.
* `/noclip` — Включить режим свободного полета сквозь блоки.
* Полный список можно найти в меню **Help & Controls -> Commands Encyclopedia**.
